# Weekly Fitness Tracker (P4E Intermediate Python Project)
 
This project combines nutrition tracking, gym programming, budgeting, savings tracking, and workout volume analysis into one structured program.

---

# 📌 Features

## 🏋️ Fitness & Nutrition Tracking
- Weekly calorie surplus/deficit analysis
- Protein intake evaluation
- Maintenance calorie estimation
- Body-type-based food recommendations

---

## 💪 Gym Programming System
- Gym split recommendations
- Muscle group frequency tracking
- Training history analysis
- Workout suggestions for under-trained muscles

### Muscle Groups Tracked
- Chest
- Back
- Legs
- Shoulders
- Arms

---

## 🏋️ Training Volume Tracker
Tracks:
- Exercise names
- Weight/load used
- Number of sets
- Number of repetitions
- Total training volume

Uses P4E list concepts such as:
- `.append()`
- Parallel lists
- `len()`
- Iteration using `for` loops

---

## 💰 Budget & Savings System
- Weekly food budget tracking
- Overspending detection
- Savings accumulation (“bank system”)
- Remaining budget analysis
